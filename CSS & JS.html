🖥️ Languages Used in Your Website

Your MnauSheepFarm website is built using 3 basic web languages:

1️⃣ HTML

Full name: HyperText Markup Language

  • This is the structure of the website
  • Creates the pages, headings, text, buttons, forms
  • Like the skeleton of the website 🦴

2️⃣ CSS

Full name: Cascading Style Sheets

  • This is the design & colors
  • Controls fonts, colors, layout, animations
  • Like the clothes & paint of the website 🎨

3️⃣ JavaScript

Full name: JavaScript (JS)

  • This is the working features
  • Controls image upload, price editing, add/remove listings, toast messages
  • Like the brain of the website 🧠
